# DragonOS

**Languages** 中文|[English](README_EN.md)

&nbsp;

这是一个运行于x86_64平台的64位操作系统。目前正在开发之中！

## 开发环境

GCC>=8.0

bochs==2.7

bximage

目前无法在qemu虚拟机上正常运行。

## 如何运行？

1. clone本项目

2. 运行命令 bash run.sh
